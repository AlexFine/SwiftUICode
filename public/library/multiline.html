Multiline Text Field

Multiline Text Field in a production app

Multiline Text Field in isolation 


import SwiftUI
import TextView

struct ContentView: View {
  @State var text = ""
  @State var isActive = false

  var body: some View {
    TextView(
      text: $text,
      isEditing: $isActive
    )
  }
}

Get Started

MultilineTextField requires a text state, and an isActive state.

The text state stores the user entered text. The isActive state stores whether the user is actively interacting with the textfield.

First add the state variables to your view.

@State var text = ""
@State var isActive = false

Second, add the textview.

TextView(
  text: $text,
  isEditing: $isActive
)

Customization

MultilineTextField supports a variety of custom inputs. All custom inputs are built into the struct initialization.

Example: Add a placeholder, add a bolder, and change the default height with the following code block: 

MultilineTextField(
    text: $text,
    isEditing: $isActive,
    placeholder: "Enter feedback ...",
    hasBorder: false ,
    minHeight: 150 
)

A mostly exhaustive input list, and their defaults, is as follows. Import the package and view the entire custom input list in XCode.

Input Description
placeholder: String? = nil Add a placeholder to your textfield.
hasBorder: Bool = true Remove the border from the textfield. Use to either create your own custom border, or remove it altogether.
textAlignment: Self.TextAlignment = .left Text alignment.
placeholderAlignment: Alignment = .left Placeholder text alignment.
textHorizontalPadding: CGFloat = 10 Horizontal text padding.
textVerticalPadding: CGFloat = 15 Vertical text padding.
placeholderHorizontalPadding: CGFloat = 15 Horizontal placeholder padding.
placeholderVerticalPadding: CGFloat = 15 Vertical placeholder padding.
font: UIFont = Self.defaultFont Font.
textColor: UIColor = .label Text color.
placeholderColor: Color = .init(.placeholderText) Placeholder color.
backgroundColor: UIColor = .clear Background color.
contentType: ContentType? = nil NA.
autocorrection: Autocorrection = .default Custom autocorrection.
autocapitalization: Autocapitalization = .sentences Custom autocapitalization.
isEditable: Bool = true Allow users to edit the textfield.
isSelectable: Bool = true Allow users to select the textfield.
isScrollingEnabled: Bool = true Can users scroll within the textfield when applicable.
