Get Started

LineChart requires an array of doubles, denoted data, to serve as its input.

First add the data variable to your view.

@State var data = [1,3,2]

Second, add the LineChart.

LineChart(data: data)

Customization

LineChart supports a variety of custom inputs. All custom inputs are built into the struct initialization.

Example: Add a title, change the style, and display chart statistics with the following code block: 

LineChart(
    data: self.data,
    title: "My Graph",
    style: .tertiary,
    displayChartStats: true
)

An exhaustive input list, and their defaults, is as follows. Import the package and view the entire custom input list in XCode.

Input Description
title: String? = nil Add a title above your graph.
subTitle: String? = nil Add a subtitle below your title and above your graph.
style: LineChartStyle? = .primary LineChartStyle is an enum of different style choices. You can always create a custom style with .custom
curvedLines: Bool = true Does your graph have curved or straight lines.
cursorColor: Color = Colors.IndicatorKnob When a user clicks on the graph, what color is their cursor.
displayChartStats: Bool = false Show text that reads the final data point, the current selected point (when applicable) and the percent change from the first data point.
width: CGFloat = 360 Chart width.
height: CGFloat = 420 Chart height.
titleFont: Font = .system(size: 30, weight: .regular, design: .rounded), Title font.
subtitleFont: Font = .system(size: 14, weight: .light, design: .rounded), Sub-Title font.
dataFont: Font = .system(size: 16, weight: .bold, design: .monospaced), Data font.
floatingPntNumberFormat: String = "%.1f" A regular expression for how to format datapoint from the chart.
fullScreen: Bool = false Allow chart view to expand to the entire screen â€“ including safe areas.
