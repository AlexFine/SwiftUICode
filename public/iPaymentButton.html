iPaymentButton

Quickly implement & easily customize Apple's required payment button. üíµ

1.

Install iPaymentButton.

Instructions

(1/3) Open XCode. Go to File > Swift Packges > Add Package Dependency...

(2/3) Input https://github.com/benjaminsage/iPaymentsButton.git & click Next.

(3/3) Select Version: Up to Next Major & click finish.

2.

Add iPaymentButton to your project. 

import SwiftUI
import iPaymentButton

struct ContentView: View {
    var body: some View {
        iPaymentButton(action: {
            // Add your custom payment code here 
        })
    }
}

3.

Customize your iPaymentButton.


Examples

Starter

Use our built-in applePayDemo() to demo a purchase experience.

import SwiftUI
import iPaymentButton

struct ContentView: View {
    var body: some View {
        iGraphicsBox().stack(3)

        iPaymentButton(action: {
            iPaymentButton.applePayDemo()
            // Add your custom payment code here 
        })
    }
}

Modern

Change the button type and style. 

import SwiftUI
import iPaymentButton

struct ContentView: View {
    var body: some View {
        iGraphicsView(.first)

        iPaymentButton(type: .support, style: .whiteOutline, action: {
            iPaymentButton.applePayDemo()
            // Add your custom payment code here 
        })
    }
}

Customize

iPaymentButton has one required parameter: an action to execute on button tap. iPaymentButton supports several custom initalizers and one custom modifier.

Example: Change the text, style and corner radius with the following code block: 

iPaymentButton(type: .support, style: .whiteOutline, action: { } )
    .cornerRadius(25)
