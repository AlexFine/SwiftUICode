Package Import Instructions

All packages use the Swift Package Manager for installation. You can find Apple's official documentation here.
We standardized all SwiftUICode package implementation to make startup easier.




Package Import

  1. (1/5) Open XCode. Go to File > Swift Packges > Add Package Dependency...
  2. (2/5) Select your current project
  3. (3/5) Input download URL & click Next.
  4. (4/5) Choose Branch: master
  5. (5/5) Click finish
